# 🧠 Mind Maze – Python Game

A fun and interactive maze-based Python game where players navigate through a maze, solve challenges, and test their logical thinking skills.

---

## 🚀 Features

- 🎮 Interactive maze gameplay  
- 🧩 Logic-based challenges  
- 🧭 Maze navigation system  
- 💡 Improves problem-solving skills  
- 🐍 Built using Python

## ⚙️ Requirements

- Python 3.x

(Optional) Install dependencies:

pip install -r requirements.txt

---

## ▶️ How to Run

1. Clone the repository:

git clone https://github.com/Sampada-26/Mind_Maze.git

2. Go to the project folder:

cd Mind_Maze

3. Run the game:

python main.py

---

## 🎯 Gameplay

- Use keyboard keys to move inside the maze  
- Find the correct path  
- Reach the goal to win the game
